Exercícios de JavaScript

Exercício 1

Crie um programa que o usuário digite uma frase e que irá imprimir ela 5 vezes.

Exercício 2

Escreva um programa que imprima a tabuada de 7 (de 1 a 10).

Exercício 3

Escreva um programa que receba uma string e imprima cada caractere em uma linha separada.

Exercício 4

Escreva um programa que conte quantas vogais (a, e, i, o, u) há em uma string fornecida pelo usuário.

Exercício 5

Escreva um programa que conte quantas vezes uma determinada letra ocorre em uma string.
